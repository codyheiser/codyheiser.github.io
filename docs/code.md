## Packages

* [`kitchen`](https://github.com/codyheiser/kitchen): automated manipulation and processing of <acronym title="single-cell RNA sequencing">scRNA-seq</acronym> counts files from command line ([documentation](https://codyheiser.github.io/kitchen/))<br />[![Latest Version][tag-version-kitchen]][repo-url-kitchen]

* [`cnmf`](https://github.com/codyheiser/cnmf): packaged and streamlined version of <acronym title="consensus nonnegative matrix factorization">cNMF</acronym>, adapted from [__[Kotliar, _et al._ (2019)]__](https://github.com/dylkot/cNMF) ([documentation](https://codyheiser.github.io/cnmf/))<br />[![Latest Version][tag-version-cnmf]][repo-url-cnmf]

* [`MILWRM`](https://github.com/Ken-Lau-Lab/MILWRM): Multiplexed Image Labeling With Regional Morphology __[Kaur, _et al._ (2024) _Communications Biology_]__ ([documentation](https://ken-lau-lab.github.io/MILWRM/))<br />[![Latest Version][pypi-image-MILWRM]][pypi-url-MILWRM]

* [`dropkick`](https://github.com/KenLauLab/dropkick): Single-cell RNA-seq quality control and filtering package __[Heiser, _et al._ (2021) _Genome Research_]__ ([documentation](https://kenlaulab.github.io/dropkick/))<br />[![Latest Version][pypi-image-dropkick]][pypi-url-dropkick]
